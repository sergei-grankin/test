# test

#### How to Install

Run in terminal:
```javascript
	npm i
```
======

#### How to build

Run in terminal:
```javascript
	gulp __dev
```
======

#### How to play

1. Click "Start" button to start the game
2. Using a mouse it is necessary to place the yellow sector above the line and hold over 0,5 sec.
3. In order to go to the next level please click on a button "Start".
